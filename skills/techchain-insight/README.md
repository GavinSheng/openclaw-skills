# TechChain Insight - 科技链·热点透视

## 🚀 技能概述

**TechChain Insight** 是一个专业的科技产业链分析技能，能够：

1. **实时捕捉**科技圈突发热点新闻
2. **深度拆解**新闻背后的全产业链利益链条
3. **精准映射**至全球主要资本市场（A 股/港股/美股）的上市公司

## 📋 覆盖领域（16 个）

### 成熟领域
- 🖥️ **半导体** - 芯片设计/制造/封装/设备/材料
- 🤖 **人工智能** - AI 芯片/大模型/应用/算力
- 🔋 **新能源** - 光伏/风电/储能/氢能
- 🚗 **新能源汽车** - 电池/电机/电控/整车
- 🔋 **固态电池** - 电解质/正负极/电池厂
- 🚙 **自动驾驶** - 传感器/算法/芯片/系统

### 新兴领域
- 📡 **6G 通信** - 太赫兹/卫星互联网/空天地一体化
- ⚛️ **量子计算** - 超导/离子阱/光量子
- 🤖 **人形机器人** - 伺服/减速器/控制器
- 🚀 **商业航天** - 火箭/卫星/发射服务
- 🧬 **合成生物** - 基因/发酵/生物制造
- 🛸 **低空经济** - eVTOL/无人机/飞行汽车
- 🧠 **脑机接口** - 侵入式/非侵入式/神经接口
- ☢️ **核聚变** - 托卡马克/人造太阳
- 🌐 **元宇宙** - VR/AR/NFT/数字人
- 🔆 **钙钛矿电池** - 钙钛矿/叠层电池

## 💡 使用场景

### 场景 A：突发新闻解读
```
用户：NVIDIA 发布新一代 Blackwell 架构芯片，分析一下影响

技能：
- 搜索验证新闻真实性
- 分析 HBM 内存、CoWoS 封装、光刻机等上游供应商
- 列出下游服务器厂商
- 映射 A 股/港股/美股相关股票
```

### 场景 B：投资线索挖掘
```
用户：固态电池最近有什么突破？

技能：
- 搜索最新研报和新闻
- 分析电解质、正负极材料厂商
- 识别受益电池厂
- 给出风险提示
```

### 场景 C：事件驱动分析
```
用户：某地锂矿停产，对产业链有什么影响？

技能：
- 分析对碳酸锂价格影响
- 推导中游电池厂成本变化
- 分析下游车企毛利影响
```

## 🔧 使用方法

### 命令行调用
```bash

# 分析突发新闻

# 查询领域动态
uv run scripts/main.py "固态电池最近有什么突破"
```

## ⚠️ 约束条件

- **准确性** - 严禁幻觉，不确定信息标注"待证实"
- **时效性** - 优先 24-48 小时内信息
- **客观性** - 不提供买卖建议，仅做事实分析
- **可追溯** - 每个结论必须有来源支撑
