# TechChain Insight - 科技链·热点透视

## 核心能力

实时捕捉科技圈突发热点新闻，利用联网搜索验证信息准确性，深度拆解新闻背后的全产业链利益链条，精准映射至全球主要资本市场（A 股、港股、美股）的具体上市公司及其受益业务环节。

## 覆盖领域

- 🖥️ **半导体** - 芯片设计/制造/封装/设备/材料
- 🤖 **人工智能** - AI 芯片/大模型/应用/算力
- 🔋 **新能源** - 光伏/风电/储能/氢能
- 🚗 **新能源汽车** - 电池/电机/电控/整车
- 🚙 **自动驾驶** - 传感器/算法/芯片/系统

## 使用方法

### 命令行调用
```bash
# 分析突发新闻
uv run scripts/main.py "NVIDIA 发布新一代 Blackwell 架构芯片"

# 查询领域动态
uv run scripts/main.py "固态电池最近有什么突破"

# 分析事件影响
uv run scripts/main.py "锂矿停产对产业链影响"
```

### 技能触发
直接询问相关领域问题，自动触发分析。

## 核心功能

1. **信息获取与验证** - 多源搜索、交叉验证、实体提取
2. **产业链图谱构建** - 纵向穿透、横向竞争分析
3. **资本市场映射** - 全球标的匹配、受益逻辑、风险警示
4. **结构化报告生成** - Markdown 格式、引用标注

## 约束条件

- ⚠️ **准确性** - 严禁幻觉，不确定信息标注"待证实"
- ⚠️ **时效性** - 优先 24-48 小时内信息
- ⚠️ **客观性** - 不提供买卖建议，仅做事实分析
- ⚠️ **可追溯** - 每个结论必须有来源支撑
